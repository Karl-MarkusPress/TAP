Rational Rose XDE

Rational Rose XDE on täiustatud tarkvaraarendustööriist, mis on integreeritud Microsoft Visual Studio .NET ja Rational Application Developeriga. IBMi Rational Software osakonna poolt välja töötatud, asendati 2006. aasta juunis Rational Rose'i perekonnaga.

Funktsioonid

Rational Rose'i komplekt hõlmas tarkvara kujundamise UML-i modelleerimistööriistu, mis lihtsustasid tarkvara kavandamise protsessi, pakkudes ehitamiseks selget kavandit. See pakkus selliseid funktsioone nagu allikapõhine pöördprojekteerimine ja sujuv integreerimine pärandkeskkondadega.

Järglased

Sellised järglased nagu Rational Software Architect ja Rational Software Modeler edestasid Rose XDE-d UML 2.x, kohandatavate mustrite ja kaasaegsete metoodikate, nagu Service-Oriented Architecture (SOA) toega.

Üldine staatus

Kunagi turuliider olnud Rational Rose sai 2006. aastaks Rational Software Architecti esiletõusuga silmitsi pärandstaatusega. Sellest hoolimata jätkas IBM Rational Rose'i pakkumist tootena vähemalt 2014. aastani, mis ühildus Visual Studio 2005 ja Windows Vista keskkondadega.
