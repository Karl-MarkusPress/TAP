Selenium


Selenium on avatud lähtekoodiga automatiseerimisraamistik, mida kasutatakse veebirakenduste jaoks, pakkudes erinevaid tööriistu ja raamatukogusid brauseri automatiseerimise toetamiseks. Selle peamised komponendid hõlmavad Selenium IDE-d, Seleniumi kliendi API-d, Seleniumi kaugjuhtimist (RC), Seleniumi WebDriverit ja Seleniumi võrku.

Selenium IDE

Täielik integreeritud arenduskeskkond Seleniumi testide jaoks, mis võimaldab funktsionaalsete testide salvestamist, redigeerimist ja silumist. Skriptid kirjutatakse Selenese'is, erilises testimisskriptimise keeles Seleniumi jaoks.

Seleniumi kliendi API

Pakub liideseid testide kirjutamiseks erinevates programmeerimiskeeltes nagu Java, C#, Python jne, võimaldades suhtlemist Seleniumiga.

Seleniumi kaugjuhtimine (RC)

Server, mis võtab brauserile käske vastu HTTP kaudu. See võimaldab automatiseeritud testide kirjutamist erinevates programmeerimiskeeltes ja paremat integreerimist olemasolevate üksuse testimisraamistikega.

Seleniumi WebDriver

Seleniumi RC järglane, see on liides brauseri automatiseerimisskriptide kirjutamiseks, mis töötavad erinevates brauserites. WebDriver käivitab brauseri eksemplarid otse ja juhib neid, ilma et oleks vaja spetsiaalset serverit.

Seleniumi võrk

Server, mis hõlbustab testide käivitamist kaugjuhitavatel brauseri eksemplaridel, võimaldades paralleelset testimist mitmel masinal ja erinevate brauseriversioonide ja konfiguratsioonide haldamist keskselt.
